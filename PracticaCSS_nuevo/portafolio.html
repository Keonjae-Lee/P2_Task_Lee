Sobre este portafolio

Esta página documenta mi trayectoria de aprendizaje en el área de Desarrollo Web. A través de la materia de Fundamentos de Desarrollo Web en la ESPE, he ido adquiriendo conocimientos sobre estructura HTML, estilos con CSS, semántica web y organización de proyectos.

El objetivo de este portafolio es reunir en un solo lugar las habilidades desarrolladas, los conceptos aprendidos y el proyecto práctico realizado durante el semestre.

Materia: Fundamentos de Desarrollo Web. Facultad de Ciencias Ambientales, ESPE - 2026

Habilidades adquiridas

HTML Semántico

Uso de etiquetas semánticas como header, nav, main, section, article y footer para organizar el contenido de forma clara y accesible.

Hojas de estilo CSS

Aplicación de selectores de etiqueta, clase e ID. Uso de propiedades de color, tipografía, espaciado, bordes y diseño con flexbox y grid. Estilos en archivos externos por página.

Proyecto integrador

Construcción de un sitio web con múltiples páginas enlazadas, rutas relativas, carpetas organizadas, formularios funcionales, multimedia y diseño visual coherente.

Proyecto integrador del semestre

Como parte de la materia se desarrolló el proyecto PracticaCSS, un sitio web de práctica que integra los conceptos de HTML semántico y CSS vistos en clase.

PracticaCSS - Sitio web de práctica

El proyecto consiste en un sitio web con varias páginas enlazadas entre sí, organizadas con rutas relativas y carpetas separadas para imágenes, audio, video, páginas internas y archivos CSS.

Estructura del proyecto: index.html, carpetas css/, img/, pages/, audio/ y video/.

Próximos pasos

A partir de lo aprendido en esta materia, los siguientes objetivos de aprendizaje son:

Profundizar en CSS avanzado

Aprender animaciones CSS, variables (:root), pseudo-elementos (::before, ::after) y diseño responsive completo con media queries.

Introducción a JavaScript

Añadir interactividad a las páginas web: validaciones del lado del cliente, manipulación del DOM y consumo de APIs básicas.

Publicar en internet

Utilizar servicios como GitHub Pages o Netlify para publicar proyectos web de forma gratuita y compartir el portafolio con otros.
